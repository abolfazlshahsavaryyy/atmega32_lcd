# ATmega32 LCD Project

This project demonstrates how to display text on a 16x2 LCD using ATmega32 microcontroller and CodeVisionAVR. The text can be displayed all at once or letter by letter with delay.

---

## How to Run the Project

### 1. Clone the Project

Open your terminal or Git Bash and run:

```bash
git clone https://github.com/abolfazlshahsavaryyy/atmega32_lcd.git
```

### 2. Open Proteus File

Navigate to the src folder inside the cloned project.

Open the Proteus project file: MoverLCD.pdsprj (or the file with the Proteus extension you have).

### 3. Select the HEX File

In Proteus, right-click the ATmega32 microcontroller.

Select Edit Properties â†’ Program File.

Choose the compiled HEX file from the src\CodeVisionAVR\Debug\Exe folder (generated by CodeVisionAVR).

Example path: src\CodeVisionAVR\Debug\main.hex

### 4. Run the Project

Click the Play button in Proteus.

The LCD should display the text.
